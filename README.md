# epidemic

此项目是vue+typescript+echarts构建的疫情后台管理系统，本项目目前只是完成了前端页面的构建，尚未接入后台数据

### 技术栈

------

- vue全家桶

- echarts

- element-ui

### 项目运行

------

```
npm run serve
```
